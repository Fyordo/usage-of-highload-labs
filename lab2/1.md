### Отключение модуля cdrom через blacklist

```
vboxuser@Ubuntu:~/Desktop$ sudo nano /etc/modprobe.d/blacklist-cdrom.conf
blacklist cdrom
blacklist sr_mod
```

```
vboxuser@Ubuntu:~/Desktop$ sudo update-initramfs -u
update-initramfs: Generating /boot/initrd.img-6.14.0-32-generic
```

```
vboxuser@Ubuntu:~/Desktop$ sudo reboot
```

### Конфигурация ядра, параметр `CONFIG_XFS_FS`

```
vboxuser@Ubuntu:~/Desktop$ ls -l /boot/config-6.14.0-32-generic
-rw-r--r-- 1 root root 296118 Sep  2 12:25 /boot/config-6.14.0-32-generic
```

```
vboxuser@Ubuntu:~/Desktop$ grep "CONFIG_XFS_FS" /boot/config-6.14.0-32-generic
CONFIG_XFS_FS=m
```

`CONFIG_XFS_FS=m` означает, поддержка XFS реализована как загружаемый модуль ядра.
